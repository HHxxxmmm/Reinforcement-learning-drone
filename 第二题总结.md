> **定稿模型**：`model/junior_q2_phase3/ppo_junior_p3_1000_steps.zip`  
> **代码目录**：`Reinforcement-learning-drone/`  
> **最后更新**：2026-07-01

## 2. 思路演变总览

```mermaid

    subgraph q2 [第二题主线]
        P1[P1 pitch+油门<br/>锁 yaw y=0] -->|1k ckpt| P2a[P2 初版 max_yaw=0.15<br/>reset yaw head]
        P2a -->|1k 3/3| P2b[P2 v2 无 max_yaw<br/>全量扫盘]

    end
    Q1[Q1 simple 经验] -.->|yaw reset 早停 y扫点| P2a
    PROBE[探针 trim/掉高] -.->|policy_init| P1
```

### 2.1 四次范式转变

| 次序 | 从 | 到 | 触发原因 |
|------|----|----|----------|
| **① 控制维** | 照搬 Q1 锁 pitch | **pitch + 油门** | pitch=0 配不平，持续掉高 |
| **② 初始化** | PPO 默认 thr≈0.23 | **`policy_init.junior_p1`** | 探针：配平需 thr≈0.50 |
| **③ 续训节奏** | 长训 60k | **1k 存盘 + eval 早停** | 7k 能杀、20k 坍缩（P1/P2 共性） |
| **④ P2 hand-off** | 直接解锁 yaw | **reset_yaw_head_on_load** | P1 锁 yaw → action[3] 漂移（+0.31 右偏） |

### 2.2 时间线（报告用）

| 阶段 | 关键决策 | 结果 | 定稿 ckpt |
|------|----------|------|-----------|
| **P1** | v1–v8 迭代、x 轴 overshoot、approach_slow | y=0 **3/3**，~280 m/s 俯冲 | **p1@1k** |
| **P2 初版** | reset yaw + max_yaw=0.15 | y=0 **3/3**，169 步 | p2@1k |
| **P2 v2** | 解除 max_yaw，12 ckpt 扫盘 | **1k 唯一全指标最优** | **p2_v2@1k** |
| **P3** | y 随机 + +y 0.70，lr 3e-5 | **~1k 步断连早停** | **p3@1k 11/11** |

## 4. Phase-2：解锁 yaw

> 详见 `第二题P2总结.md`

### 4.1 P2 初版

- P1@1k + **`reset_yaw_head_on_load: true`**
- `max_yaw: 0.15`，`yaw_misalign_weight: 8`
- **ppo_junior_p2_1000**：3/3，yaw −0.29°

### 4.2 P2 v2（解除 max_yaw）

- 从 P2@1k 续训，**不 reset yaw**
- 12 ckpt × 3 局 eval（room 20411）

| ckpt | kills | pitch° | dz_m | 备注 |
|------|-------|--------|------|------|
| **1000** | **3/3** | **+2.9** | **+0.4** | **定稿** |
| 2000–6000 | 3/3 | +6~+8 | +3~+11 | 可杀但劣化 |
| 3000+ | 0/3 | +27~+30 | +86~+184 | pitch 饱和爬飞 |

**P2 定稿**：`ppo_junior_p2_v2_1000_steps.zip`

## 6. 定稿设计与实现细节

### 6.1 Agent

| 项 | 定稿 |
|----|------|
| 算法 | PPO，`MlpPolicy`，128-128 |
| 观测 | 20 维（同 Q1） |
| 动作 | throttle + pitch + yaw（roll 锁） |

### 6.2 动作映射（定稿）

| 维 | 范围 | 说明 |
|----|------|------|
| throttle | [0, **0.46**] | 物理上限控速 |
| pitch | ±**0.35** rad | 配平/俯冲 |
| yaw | ±**0.5** rad（scale=0.5，无 max_yaw） | P2 v2 起无额外 cap |
| roll | 0 | 锁定 |

### 6.3 Reward 要点（junior 特有）

| 分量 | 作用 |
|------|------|
| `altitude_match` / `altitude_below_penalty` | 同高接敌 |
| `approach_slow_penalty` | 远距防慢爬 |
| `speed_cruise_penalty` | 远距限速（门控） |
| `attack_speed_bonus` / `attack_dwell` | 盒内正奖励 |
| `yaw_misalign_penalty` | P2 起 weight=8 |
| `enemy_damage` / `kill_bonus` | HP 直反馈 |

### 6.4 截停

- overshoot（x 轴 fallback，margin 45 m）
- altitude_miss（接敌段 x<450 m 且掉高 >220 m）
- 512 步上限

## 9. 踩坑清单

| 现象 | 原因 | 处理 |
|------|------|------|
| pitch=0 掉高 | Junior \(C_L(\theta)\) | 解锁 pitch；探针配平 |
| 未训 PPO thr≈0.23 | 默认动作均值 | **`policy_init.junior_p1`** |
| P2 解锁右拐 | P1 lock yaw 漂移 | **reset_yaw_head** |
| 续训 pitch→+30° | PPO 过更新 + 爬飞局部最优 | **1k 早停** |
| v2 3k+ 0 杀 | dz 86~184 m 飞过靶 | eval 扫盘选 1k |
| Q1 ckpt 不可 load | 动力学不同 | Junior **从零 P1** |
| P3 不必长训 | 1k 已 11/11 | 峰值即停，避免坍缩 |

## 10. 实验心得与报告要点

### 10.1 相对第一题

1. **配平是第一性**：Junior 必须先 pitch+油门，再 yaw，再 y 随机。  
2. **同一范式复用**：锁维漂移、1k 早停、y 扫点 eval 在 Q1/Q2 均成立。  
3. **Junior P3 更顺**：因 P2@1k pitch/yaw 已稳，**1k 步 P3 即 11/11**；Q1 需 p31k_resume 多轮。  
4. **探针驱动设计**：`probe_junior_altitude/trim` 直接决定 policy_init 与 throttle 上限。

### 10.2 报告建议结构

1. Simple vs Junior 动力学差异（`../模型说明`）  
2. 课程设计 P1/P2/P3 与 Q1 对照  
3. 探针 → policy_init → reward 迭代（P1 v1–v8）  
4. yaw reset 与 P2 v2 扫盘数据  
5. P3 y 扫点 11/11 与轨迹指标（pitch/yaw/dy/dz）  
6. 「峰值早停」作为两题共同结论
