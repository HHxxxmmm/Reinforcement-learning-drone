## 1. 与第一题的核心差异

| 维度 | Q1（simple） | Q2（junior） |
|------|-------------|--------------|
| 升力 | \(C_L\) 常数，\(F_L \propto v^2\) | **\(C_L(\theta)\)** 随迎角/pitch 变 |
| 高度 | pitch 锁 0 仍可平飞 | **pitch=0 往往 \(F_L < Mg\)** → 持续掉高 |
| 油门语义 | mainly 加速/减速（几乎无刹车） | 油门 ↔ 速度 ↔ 升力 **三者耦合** |
| Q1 控维 | 最终 **油门 + yaw**（pitch/roll 锁） | P1 需 **pitch + 油门**（yaw/roll 锁） |
| 典型末速 | ~100–130 m/s 可接敌 | 易 **~280 m/s** 接敌击杀，或飙至 **350–430 m/s** 穿靶 |

**设计原则（Junior）**：

> **pitch 管升力/高度，油门管速度；不要指望 Q1「小油门接敌」直接迁移。**

参考：`../模型说明`（Simple vs Junior 公式对照）。

## 2. P1 任务设计（相对 Q1 课程）

类比 Q1 的「锁 y、锁 yaw 只学油门」，Q2 P1 在 **固定初始几何** 下尽量 **去除侧向与偏航影响**：

| 项 | P1 设定 |
|----|---------|
| 初始化 | 我机 `(0,0,100)`，敌 `(120,0,100)`，`enemy_y_range: [0,0]` |
| 初速 | 11 unit → **110 m/s** |
| 锁定 | **roll + yaw**（`fixed_yaw=0`） |
| 学习 | **pitch + 油门** |
| 不引入 | y 随机、yaw 转向（留 P2/P3） |

**P1 子目标分解**：

1. 配平：同高度接敌（`altitude_match`）  
2. 接敌：对准 + 走廊/中线  
3. 击杀：攻击盒内打穿 hp

## 3. 探针实验与关键结论

### 3.1 固定油门探针（`scripts/probe_junior_altitude.py`）

**条件**：pitch/yaw 锁 0，junior 房间。

| 油门 | 150–200 步 Δz | 结论 |
|------|----------------|------|
| 0.12（Q1 接敌） | **−429 m** | 严重掉高 |
| 0.00 | **−557 m** | 更快坠 |
| 0.35 | **−57 m** | 仍掉，但慢 |
| 0.50, pitch=0 | **+6 m** | 接近配平区 |
| 0.65, pitch=+0.15 | **+23 m** | 爬升过快 |

**结论**：junior 下 **pitch=0 时普遍掉高**；配平需 **pitch 和/或 更大油门**。

### 3.2 Trim 扫掠（`scripts/probe_junior_trim.py`）

| 工况 | 趋势 |
|------|------|
| thr=0.50, pitch=0 | 略爬，接近平飞 |
| thr=0.65, pitch=+0.15~0.28 | 明显爬升 |

**结论**：**pitch 调 \(C_L\)/配平**，油门主要调速度；P1 必须解锁 pitch。

### 3.3 PPO 初值与「油门像没工作」（关键）

未训练 PPO 的 action 均值 ≈ 0 → 映射后 **throttle ≈ max/2（~0.23）**。  
在 junior 下 **0.23 + pitch≈0 远不够配平**（探针需 ~0.50），表现为：

- 速度卡在 **~110 m/s**（初速附近）  
- **持续掉高**（`altitude_below_penalty` 很大）  
- 日志里 `thr=0.12~0.29` — **并非 lock_throttle**，而是策略输出偏低  

**处理**：`policy_init.junior_p1: true`（`utils/policy_reset.py`）把 throttle/pitch bias 设到探针配平区（thr≈0.40，pitch_cmd≈+0.12 rad）。

## 4. 配置与 reward 迭代时间线

| 版本 | 思路 | 结果 |
|------|------|------|
| **v1** | 照搬 Q1 P1：锁 pitch，只油门 | **持续掉高**，不可用 |
| **v2** | 提初速 13、`max_throttle` 0.65、解锁 pitch | 高度改善，**~400 m/s** |
| **v3–v6** | 速度罚/物理限速/盒内正奖励迭代 | hp 卡 **0.10–0.12**，难击杀 |
| **v7** | `max_throttle=0.46`、宽速度带 + 低 hp bonus | 高度稳住，仍 **~350 m/s** |
| **v8 方案 A** | 收紧 below_penalty、gate=15、`max_throttle=0.42` 等 11 项 | 7k 步曾 **~280 m/s + 击杀**；续训易 **坍缩**（~100 m/s 慢爬） |
| **v8-lite** | 回 v7 动作空间 + 轻度反俯冲 | 中间过渡，未作定稿 |
| **v8-A-tuned（定稿）** | v7 动作 + 截停修复 + `approach_slow` + **policy_init** + **1k 存盘** | ✅ **1k ckpt eval 3/3 击杀** |

### 4.1 范式转变（报告可写）

| 次序 | 从 | 到 | 触发原因 |
|------|----|----|----------|
| ① 控制维 | 锁 pitch + 油门 | **pitch + 油门** | 探针：pitch=0 配不平 |
| ② 速度信号 | 重罚超速 | **物理 throttle 上限 + 盒内正奖励** | 「冲过惩罚区」局部最优 |
| ③ 高度–速度 | 单一油门 | **高度门控** + pitch 配平 | 收油掉高 vs 收速矛盾 |
| ④ 训练工程 | 5k 存盘、随机初值 | **1k 存盘 + policy_init** | 好策略在 7k 前出现、续训易坍缩 |
| ⑤ 截停 | 仅 nose-forward overshoot | **x 轴 fallback + 接敌段 altitude_miss** | 大 pitch 俯冲时跑满 512 步 |

## 5. 定稿设计（v8-A-tuned）

### 5.1 环境与动作

文件：`config/envs.junior.phase1.yaml`

```yaml
initial_speed: 11
enemy_y_range: [0, 0]
stage_params:
  overshoot_use_x_axis: true      # 锁 yaw 对头时 x 轴接敌距离 fallback
  altitude_miss_below_m: 220.0    # 低于敌机 >220m 截停
  altitude_miss_range_m: 450.0    # 仅 x_close<450m 时生效（远距留加速时间）
  approach_slow_penalty_weight: 0.012
  approach_slow_min_mps: 170.0
action:
  lock_roll: true
  lock_yaw: true
  lock_pitch: false
  max_throttle: 0.46
  max_pitch: 0.35
  action_scale: 0.50
```

### 5.2 Reward 要点

| 分量 | 作用 |
|------|------|
| `altitude_match` / `altitude_below_penalty` | 同高接敌 |
| `approach_slow_penalty` | 远距（660–900m）低于 170 m/s 罚 → 防「慢爬」 |
| `speed_cruise_penalty` | 远距限速（高度门控内，cap 220 m/s） |
| `attack_speed_bonus` / `attack_dwell` | 盒内速度带正奖励 |
| `low_hp_attack_bonus` | hp<0.25 补刀 |
| `attack_speed_penalty` | **0** |

### 5.3 算法

文件：`config/algs.junior.phase1.yaml`

| 项 | 值 |
|----|-----|
| PPO | 128-128，60k steps |
| `save_freq` | **1000**（截获早期击杀 ckpt） |
| `policy_init.junior_p1` | **true**（配平起点偏置） |
| 前缀 | `ppo_junior_p1` |

### 5.4 相对 Q1 仓库新增/改动

| 文件 | 新增/改动 |
|------|-----------|
| `utils/reward.py` | junior 高度/速度项、`approach_slow_penalty` |
| `utils/policy_reset.py` | `reset_policy_junior_p1_head` |
| `envs/train_env.py` | x 轴 overshoot、`altitude_miss`、combat 日志含 pitch/thr/alt |
| `main.py` | 新训时 `policy_init.junior_p1` |
| `config/envs.junior.phase1.eval.yaml` | 独立 eval 房间 |
| `scripts/probe_junior_*.py` | 掉高 / trim 探针 |

## 6. 训练现象与过关 eval

### 6.1 过关前典型好 run（v8 方案 A，~7.6k steps）

```
speed=278m/s  pitch=+0.157  thr=0.16  alt=1081m  enemy_hp=0.100  kill_bonus>0
ep_rew≈+3500  ep_len≈173
```

策略模式：**先爬升 → ~280 m/s 俯冲接敌 → 击杀**。续训过久易坍缩为 ~100 m/s 慢爬 + 掉高。

## 7. 踩坑清单

| 现象 | 原因 | 处理 |
|------|------|------|
| pitch=0 掉高 | junior \(C_L(\theta)\) | 解锁 pitch；探针定配平 |
| 油门像没工作 | 未训 PPO thr≈**0.23** 不够配平 | **`policy_init.junior_p1`** |
| 速度 ~114 m/s 慢爬 | 同上 + 方案 A 收太紧 | v8-A-tuned + `approach_slow_penalty` |
| 跑满 512 步 | 大 pitch 俯冲时 `forward_distance` 不触发 overshoot | **`overshoot_use_x_axis`** |
| 远距 altitude_miss | 200m 掉高在 x=600m 就截停 | **`altitude_miss_range_m: 450`** |
| v8 方案 A 7k 能杀、20k 不能 | PPO 稀疏击杀 + value 漂移 | **1k 存盘 + 早停 eval** |
| 续训/simple 模型 load | 动力学不同 | junior **从零训**，不 load Q1 |
| 攻击盒重罚超速 | 「冲过惩罚区」 | 改正奖励 + 物理 throttle 上限 |
| P2 解锁 yaw 左拐 | P1 锁 yaw 时 action[3] 无梯度 | P2 用 **`reset_yaw_head_on_load`**（见 P2） |

## 10. P2/P3 展望

与 Q1 对称的课程（**固定初始 y，逐步解锁**）：

| 阶段 | 锁定 | 学习 | 泛化 | 状态 |
|------|------|------|------|------|
| **P1** | roll, yaw | pitch + 油门 | y=0 | ✅ 过关（1k ckpt） |
| **P2** | roll | pitch + 油门 + **yaw** | y=0 | 待做 |
| **P3** | roll | pitch + 油门 + yaw | **y∈[-5,5]** | 待做 |

### 10.1 P2 计划（下一步）

1. 新建 `envs.junior.phase2.yaml` / `algs.junior.phase2.yaml`  
2. **`lock_yaw: false`**，`max_yaw: 0.15~0.20`，`yaw_misalign_weight: 6~10`  
3. **`load_path: ppo_junior_p1_1000_steps.zip`**  
4. **`reset_yaw_head_on_load: true`**（Q1 P2 左拐教训）  
5. PPO 宜保守：`lr≈5e-5`，`n_epochs: 3`（参考 Q1 P2 v2）  
6. 过关：eval y=0，**3/3 击杀**，步数 ~170，yaw 勿长期大偏  

### 10.2 P3 计划

- 在 P2 稳定 ckpt 上解锁 `enemy_y_range: [-5, 5]`  
- 可参考 Q1 `+y` 偏置采样与 y 扫点 eval
