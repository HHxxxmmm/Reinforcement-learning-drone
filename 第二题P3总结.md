## 1. P3 任务设计

| 项 | P2 定稿 | P3 |
|----|---------|-----|
| 敌机 y | 0 | **[-5, 5]**（50 m 侧向，1 unit=10 m） |
| y 采样 | — | **`enemy_y_positive_prob: 0.70`**（过采样 +y） |
| 动作 | pitch + 油门 + yaw | **不变** |
| 奖励 / 截停 | v8-A-tuned 全套 | **继承 P2**，无额外 tweak |

**动机**：Q1 P3 曾出现「-y 全杀、+y 不转向」；Junior P2@1k yaw 已很小（≈−1.3°），且 pitch 配平稳定，预期 P3 只需 **短续训 + 早停** 即可泛化。

### 2.1 早期训练信号（iter 1–2，~1k 步内）

```
kill_bonus_mean > 0    ep_len ≈ 171–201    ep_rew ≈ 2800–3280
enemy_y 采样可见 ±3, +4 等侧向局
```

续训初期已有击杀与 damage，说明 P2 策略对侧向靶 **可迁移**；与 Q1「P3 长训后左偏」不同，Junior P3 **1k 即 peaks**。

### 3.2 观察

| 现象 | 解读 |
|------|------|
| yaw ≈ **−2.8°** 全 y 几乎不变 | 轻微左偏，但 **±50 m 侧向均能击杀**（Q1 曾 +y 全挂） |
| pitch **+1.2°~+1.9°** | 远低于 P2 续训坍缩时的 +30°，配平健康 |
| dz **−1.4~−3.4 m** | 相对初始高度 1000 m 仅小幅偏低，可接受 |
| dy **≈ −9 m** | 接敌末略有左向位置偏移，未出攻击走廊 |
| 首次扫点 +5 hp=0.12 | 偶发未杀；**复测 11/11** → 以复测为准定稿 |

## 4. 与 Q1 P3 的对比

| 维度 | Q1 P3（simple） | Junior P3 |
|------|-----------------|-----------|
| 控制维 | 油门 + yaw（pitch 锁） | **pitch + 油门 + yaw** |
| P2→P3 起点 | p2_v2@5k | **p2_v2@1k** |
| Q1 难点 | +y 不转向、左偏 | Junior **1k 即 11/11** |
| 训练量 | 20k~31k + 多轮 resume | **~1k 步**（断连早停） |
| 定稿 K | 11/11（p31k_resume final） | **11/11（p3@1k）** |
| yaw 随 y | 定稿约 −3° 且随 y 略变 | ≈−2.8° **几乎不随 y 变**，仍全杀 |

**结论**：Junior 因 P1 配平 + P2 早停定稿，P3 **无需 Q1 式多轮短续 resume**；**峰值早停** 范式在第二题再次验证。

## 5. 定稿配置

### 5.1 环境（`config/envs.junior.phase3.yaml`）

```yaml
enemy_y_range: [-5, 5]
enemy_y_positive_prob: 0.70
yaw_misalign_weight: 8.0
# altitude / speed / attack 同 P2 v8-A-tuned
action:
  lock_roll: true
  lock_pitch: false
  lock_yaw: false
  max_throttle: 0.46
  max_pitch: 0.35
  action_scale: 0.50
```

### 5.2 算法（`config/algs.junior.phase3.yaml`）

```yaml
load_path: ./model/junior_q2_phase2_v2/ppo_junior_p2_v2_1000_steps.zip
reset_yaw_head_on_load: false
total_timesteps: 20000   # 计划；实际 ~1.5k 断连
save_freq: 1000
learning_rate: 3.0e-5
n_epochs: 3
```

### 5.3 推荐 checkpoint

| 路径 | 说明 |
|------|------|
| **`model/junior_q2_phase3/ppo_junior_p3_1000_steps.zip`** | **P3 / 第二题定稿** |
| `model/junior_q2_phase2_v2/ppo_junior_p2_v2_1000_steps.zip` | P2 定稿（P3 父 ckpt） |

## 6. 踩坑清单

| 现象 | 原因 | 处理 |
|------|------|------|
| 训练 ~1.5k 断连 | UE / 网络 `ConnectionResetError` | **1k ckpt 已够**；eval 定稿后不必强续 20k |
| 首次 +5 hp=0.12 | 随机性 / UE 状态 | 复测；或固定 seed 多局 |
| 不宜长训 P3 | 与 P1/P2 相同，续训易 pitch 爬飞 | **1k 存盘 + y 扫点 + 峰值即停** |
| yaw 不随 y 变化 | 策略偏「固定微左偏」 | 当前 K=11/11 可接受；报告可讨论 |
